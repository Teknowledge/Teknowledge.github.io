Syntax Guide

print

Write to the screen

print("hi")
print("multiple", "things")
print("numbers too", 42, -8)

variables

store a piece of information

x = 1
print("x is", x)
name = "Cody"
print("hi " + name + "!")

input

gets user input - pauses program until user presses ENTER, returns anything that they typed *as a string*

name = input("Enter your name: ")
print("Your name is", name)
input("Press enter to continue...")

if statement

do this code if this condition is true

print("what is 1 + 1?")
answer = input("> ")

  print("right!")
elif (answer == "two"):
  print("clever!")

  print("wrong!")

functions

a defined group of code that can be called multiple times and optionally with arguments and with a return value

def f():
  print("I am in a function, f!")
  print("multiple lines of code grouped together!")

print("I am before the function call to f")
f()
print("I am after the function call to f")

def gimmeWahoos(x):
  print("I am in the function gimmeWahoos")
  print("I recieved an argument:", x)
  print("I am going to return the string 'wahoo' repeated x times")
  return "wahoo" * x

print("I am calling function gimmeWahoos")
print("with an argument of the integer 5")
returnValue = gimmeWahoos(5)
print("gimmeWahoos returned to me the value:", returnValue)
